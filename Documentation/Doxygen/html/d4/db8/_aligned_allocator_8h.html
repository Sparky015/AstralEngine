void * Core::AlignedAlloc ( size_t size,
uint8 alignment )
nodiscard

Allocates a memory block which has the given alignment.

Parameters
sizeThe size of the memory block to allocate
alignmentThe alignment of the allocated memory block. Must be a power of two.
Returns
A pointer to the allocated memory block. Returns nullptr if size is zero.
Warning
The max alignment support through this function is 255.
Thread Safety
This function is not thread safe.
